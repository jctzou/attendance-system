# 考勤記錄頁面規格書 (Attendance Record Spec)

> **版本**: v1.5
> **狀態**: 已實作 (Implemented)
> **最後更新**: 2026-02-28
> **目標**: 定義「打卡記錄 (Attendance Record)」頁面之月曆檢視與資料編輯互動。

---

## 1. 頁面功能概觀與角色權限

本頁面提供日曆式的出勤記錄檢視，並允許使用者點擊進行修正（補登/修改）。
- **一般員工 (Employee)**: 僅能查看**自己**的出勤紀錄，無法切換員工篩選器。
- **管理層級 (Manager / Super Admin)**: 預設查看自己，可透過上方「員工下拉選單」切換並檢視任何員工的當月打卡細節與修訂歷程。

---

## 2. 介面佈局與資料流 (UI Layout & Data Flow)

### 2.1 頁面結構
- **操作列 (Toolbar)**:
    - 月份選擇器 (YYYY-MM)，並透過 `useSearchParams` 保持 URL 同步以便書籤或分享。
    - 員工選擇器 (僅管理員可見，顯示格式：`姓名 員編`。若員工尚未設定員編，則僅顯示姓名)。
- **日曆視圖 (Calendar Grid)**: 
    - 採用 RWD 排版。手機版為單列垂直堆疊，桌面版為七列月曆佈局 (`grid-cols-7`)。
        - **大螢幕日曆自動補位**: 桌面版需依照當月第一天為星期幾進行空白底板補位 (`padding-start`)。
        - **智慧定位**: 頁面載入完成後自動平滑捲動 (`scrollIntoView`) 至今日卡片。

### 2.2 日卡片 (Day Card) 視覺呈現規則
每個「日期」為一個獨立的卡片，背景色區分今日與週末。
- **今日 (Today)**: 藍色系高亮 (`bg-blue-50`)，帶有藍色邊框。
- **週末/例假日 (Weekend)**: 淺灰色系 (`bg-slate-100`)。

| 顯示項目 | 月薪制 (Monthly) | 鐘點制 (Hourly) |
| :--- | :--- | :--- |
| **打卡時間** | 顯示上下班時間 (HH:mm) | 顯示上下班時間 (HH:mm) |
| **異常狀態標籤** | **顯示** (遲到/早退 - 紅橘字) | **依設顯示** (受開關控制) |
| **午休時數** | **顯示** (若 > 0) | **顯示** (e.g., "休 1h") |
| **工時精確度 (v1.5)** | 兩位小數 (如 8.33h) | **兩位小數 (如 7.25h)** |
| **工作時數** | 不顯示於卡片表面 | 不顯示於卡片表面 |

## 3. 特殊頁面邏輯與狀態管理 (Local Logic & State)

點擊任何一格日卡片，將啟動下方的 **操作對話框 (`AttendanceActionDialog`)**。

### 3.1 補登與修改流程 (v1.5)
- **TimeSlotSelector 管理**:
    - 工作日期由 `AttendanceActionDialog` 強制傳入 `workDate` prop。
    - 清除時間後重新選擇時，年份/月份/日期不再跳回「今日」，始終維持原設定日期。
- **午休選項**:
    - 預設提供 `0h / 1h / 1.5h` 之按鈕選項，支援彈性調整。
    - **防呆連鎖**: 若上班晚於 12:00 或下班早於 12:00（且使用者有主動交互 `hasUserInteracted`），自動將午休歸零為 `0h`。
